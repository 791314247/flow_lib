# flow_lib

#### Description
适用于嵌入式单片机的裸机程序微库，只占用你的rom 6个字节，是的，6个字节。颠覆式的设计思维，让你写代码的时候像flow(流水)一样丝滑，让你永远不用在为delay时cpu空转而烦恼，附加的超轻便的软件定时器让你轻松实现各种定时需求，令还有信号量的配方，让你任务间的同步像诗一样写意，并且能让你逻辑程序效率提升百倍以上。
